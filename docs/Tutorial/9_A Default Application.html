A Default Application

The QCApplication preconfigures a common combination of parts needed for a web application. There probably be some chapter about deploying the thing and along the way explaining some technical details. This part of the tutorial is not finished

The features of the default application

QCMagritte consist of a large number of parts that can be combined in different ways. QCApplication and QCApplicationModel show an easy way to combine them. Subclass both to have an easy start.

A QCMagritte application uses a tree of components/pages.
